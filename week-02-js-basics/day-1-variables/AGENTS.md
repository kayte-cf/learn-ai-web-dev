# AI Usage Guide - Day 1

## Core Concepts to Understand (Don't Skip)

These must be understood, not just memorized:

1. **Variables are containers** - They hold data with a name
2. **let vs const** - `const` can't change, `let` can
3. **Types** - JavaScript has different kinds of data
4. **typeof** - How you check what type something is

## Good AI Questions

- "Why does JavaScript have both `let` and `const`?"
- "What's the difference between `'string'` and `"string"` and `` `string` ``?"
- "Why does `'5' + 3` equal `'53'` but `'5' - 3` equals `2`?"
- "What other data types does JavaScript have besides string, number, boolean?"

## Bad AI Uses

- Don't ask AI to write your exercises for you
- Don't ask "what's the answer to exercise 3?"
- Don't skip typing the code yourself

## When Stuck

1. Re-read the exercise
2. Check for typos (especially quotes and semicolons)
3. Run `node script.js` and read the error message
4. THEN ask AI: "I'm getting this error: [error]. What does it mean?"

## Type Everything

Seriously. Your fingers need to learn:
- Where the backtick key is (for template literals)
- How to type `${}` inside backticks
- The rhythm of `let name = value;`

Copy-paste teaches nothing.
