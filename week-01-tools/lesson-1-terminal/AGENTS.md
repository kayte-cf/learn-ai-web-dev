# AI Usage Guide - Day 1

## When to Use AI Today

### Good Uses
- Ask AI to explain what a command does
- Ask AI to help you remember a command you forgot
- Ask AI to explain an error message

### Examples of Good Questions
- "What does the `-la` flag do in `ls -la`?"
- "I got 'permission denied' - what does that mean?"
- "How do I list only `.txt` files?"

### Avoid Using AI For
- Don't ask AI to just give you the commands to type
- Don't skip typing commands yourself
- The goal is muscle memory, not just getting answers

## AI Ethos for This Session

Today is about **building muscle memory**. Type every command yourself, even if you could ask AI to write them.

If you're stuck, try these steps first:
1. Re-read the instructions
2. Check for typos
3. Ask your instructor

Then, if still stuck, ask AI: "I tried [command] and got [error]. What went wrong?"

## Commands to Know (Don't Just Ask AI)

By end of today, you should be able to type these from memory:
- `pwd`
- `ls`
- `cd [folder]`
- `cd ..`
- `mkdir [name]`
- `touch [filename]`
- `rm [filename]`
- `cat [filename]`
