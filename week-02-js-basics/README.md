## Setup

Make sure Node.js is installed:
```bash
node --version
# Should show v18.x.x or higher
```

Navigate to this week:
```bash
cd week-02-js-basics
```

## Running Your Code

Every script runs with Node:
```bash
node day-1-variables/starter/script.js
```

Or use the npm scripts:
```bash
npm run day1
npm run day2
# etc.
```

## What is Node.js?

Node.js lets you run JavaScript outside the browser. Instead of JavaScript only working in web pages, you can:
- Run scripts from your terminal
- Build command-line tools
- Create servers (later in the course)

## Concepts You'll Learn

### Day 1: Variables & Types
- `let` and `const` for declaring variables
- Data types: string, number, boolean
- `typeof` to check types

### Day 2: Operators
- Math: `+`, `-`, `*`, `/`, `%`
- Comparison: `===`, `!==`, `>`, `<`
- String concatenation

### Day 3: Conditionals
- `if` statements
- `else` and `else if`
- Making decisions in code

### Day 4: Functions
- Declaring functions
- Parameters and arguments
- Return values
- Why functions matter

## Tips

1. **Type everything** - Don't copy-paste. Your fingers need to learn.
2. **Experiment** - Change values, see what happens, break things.
3. **Read errors** - Error messages tell you what went wrong. Read them!
4. **console.log() is your friend** - Print values to understand what's happening.
