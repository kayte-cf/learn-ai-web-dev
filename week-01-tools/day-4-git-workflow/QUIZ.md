## Answers

<details>
<summary>Click to reveal answers (try first!)</summary>

1. `git clone <url>`

2. **Local** repositories exist only on your computer. **Remote** repositories are stored on a server (like GitHub) and can be accessed from anywhere and by anyone with permission.

3. `git push`

4. **c, e, b, d, a**
   - Edit your files
   - `git status` (see what changed)
   - `git add .` (stage changes)
   - `git commit -m "message"` (save locally)
   - `git push` (upload to remote)

5. **c) `git pull`** - This downloads and integrates changes from the remote repository
