## Answers

<details>
<summary>Click to reveal answers (try first!)</summary>

1. **pwd** = "print working directory" - shows your current location in the file system

2. **d) Both a and b** - `cd ~` goes to home directory, `cd ..` goes up one level. Both work from `/Users/yourname/Documents`

3. `ls` shows files and folders. `ls -la` shows ALL files (including hidden ones starting with `.`) in LONG format with details like permissions, size, and dates

4. `mkdir projects`

5. Takes you to your home directory (e.g., `/Users/yourname`)
